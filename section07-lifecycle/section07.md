## 📚 section07-lifecycle

### 🔄 리액트 컴포넌트의 라이프 사이클

- 🐣 **Mount** :   
  - 컴포넌트가 탄생하는 순간   
  - 화면에 처음 렌더링 되는 순간   
  ex) "A 컴포넌트가 Mount 되었다!" -> A 컴포넌트가 화면에 처음으로 렌더링 되었다.

<br>

- 🐓 **Update** :
  - Mount 이후에 컴포넌트가 다시 렌더링 되는 순간   
  - 리렌더링 될 때를 의미   
  ex) "A 컴포넌트가 Update 되었다!" -> A 컴포넌트가 리렌더링 되었다.

<br>

- 🍗 **UnMount** :
  - 컴포넌트가 화면에서 사라지는 순간   
  - 렌더링에서 제외 되는 순간을 의미     
  ex) "A 컴포넌트가 UnMount 되었다!" -> A 컴포넌트가 화면에서 사라졌다.
